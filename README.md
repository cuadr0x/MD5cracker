# MD5cracker

Python script to bruteforce MD5 hashes

## Usage
```
Usage: MD5cracker.py [options]

Options:
  -h, --help            show this help message and exit
  -w WORDLIST, --wordlist=WORDLIST
                        Wordlist to use for bruteforcing
  -s HASH, --hash=HASH  String of the hash to crack
  -f FILE, --file=FILE  File with the hash to crack
  -o OUTFILE, --outfile=OUTFILE
                        Output file witch cracked passwords
  -v, --verbose         Verbose output (SLOWER)
```
